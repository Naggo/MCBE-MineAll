# MCBE-MineAll v0.2.0
MCBE GameTest Addon

## Requirements
Enabled 'GameTest Framework'

## Commands
Use `#mineall ?` to show help.

* #mineall
* #mineall list
* #mineall on
* #mineall off
* #mineall max <maxChains: number>
* #mineall help
* #mineall ?
